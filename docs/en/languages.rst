Switch Between Languages
=========================

:link_to_translation:`zh_CN:[中文]`

The documentation is now available in two languages. Please refer to the English version if there is any discrepancy.

- English
- Chinese

You can easily change from one language to another by clicking the language link you can find at the top of every document that has a translation.

.. image:: ../_static/choose_language.png

To switch languages:

1. Look for the language link at the top of any page (e.g., ``:link_to_translation:`zh_CN:[中文]``)
2. Click on the link to switch to the translated version
3. The URL will automatically update to reflect the new language

.. note::

    Not all pages may have translations. If a translation is not available, you will be directed to the English version.
